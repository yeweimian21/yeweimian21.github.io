Posted on 2019-10-17 Edited on 2023-02-09 In Redis

 

Redis的数据类型

Redis支持五种数据类型：

1. string（字符串）

2. hash（哈希）

3. list（列表）

4. set（集合）

Redis 的 Set 是 String 类型的无序集合。集合成员是唯一的，这就意味着集合中不能出现重复的数据。

Redis 中集合是通过哈希表实现的，所以添加，删除，查找的复杂度都是 O(1)。

5. zset(sorted set：有序集合)。
