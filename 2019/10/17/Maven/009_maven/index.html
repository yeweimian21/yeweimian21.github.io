使用maven重构springmvc的项目

1. 新建Maven Project

2. 复制文件

目录结构对应关系:

SpringMVC project Maven project
src src/main/java
resources src/main/resources
WebRoot src/main/webapp

文件对应关系:

SpringMVC project Maven project
src/* src/main/java/*
resources/* src/main/resources/*
WebRoot/resources/* src/main/webapp/resources/*
WebRoot/WEB-INF/view/* src/main/webapp/WEB-INF/view/*
WebRoot/WEB-INF/web.xml src/main/webapp/WEB-INF/web.xml
WebRoot/index.jsp src/main/webapp/index.jsp

3. 修改pom.xml

4. Eclipse Tomcat运行Maven项目

将项目部署至Eclipse Tomcat，启动项目

访问地址: http://localhost:8080/mavenproject

查看项目访问url:

Maven项目右键->Properties

Web Project Settings -> Context root: [项目名]

Servers -> server.xml -> <Context path="[项目名]" />

这两个地方的[项目名]是一致的

5. Tomcat 7 插件运行Maven项目

项目右键->Run As->Run Configurations…

Maven Build -> [项目名] -> Main -> Goals: tomcat7:run

访问地址: http://localhost:8080/mavenproject
