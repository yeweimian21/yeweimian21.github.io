Write to file

Parameter description
w Open a file for writing only. If the file already exists, open the file and start editing from the beginning, and the original content will be deleted. If the file does not exist, create a new one.
a Open a file for appending. If the file already exists, the file pointer will be placed at the end of the file. In other words, the new content will be written to the existing content. If the file does not exist, create a new file for writing.
r Open the file as read-only. The pointer to the file will be placed at the beginning of the file. This is the default mode.
