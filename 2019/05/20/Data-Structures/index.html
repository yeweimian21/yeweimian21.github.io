3.排序算法的性能

排序方法 时间复杂度 空间复杂度 稳定性 最好 最差 备注
冒泡排序 O(n^2) O(1) 稳定 O(n) O(n^2) n较小时较好
直接插入排序 O(n^2) O(1) 稳定 O(n) O(n^2) 大部分元素已有序时较好
简单选择排序 O(n^2) O(1) 不稳定 O(n^2) O(n^2) n较小时较好
快速排序 O(nlogn) O(logn) 不稳定 O(nlogn) O(n^2) n较大时较好
堆排序 O(nlogn) O(1) 不稳定 O(nlogn) O(nlogn) n较大时较好
归并排序 O(nlogn) O(n) 稳定 O(nlogn) O(nlogn) n较大时较好
希尔排序 O(nlogn) O(1) 不稳定 O(n) O(ns)1<s<2,s是所选分组
