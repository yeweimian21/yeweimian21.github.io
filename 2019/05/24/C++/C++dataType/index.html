bool

By default, the integer 0,1 is used to represent the bool value.

You can use std::boolalpha to output bool type results as true or false.

When std::boolalpha is used, future bool type results will be output in true or false form, unless std::noboolalpha is used to cancel the boolalpha stream format flag.
