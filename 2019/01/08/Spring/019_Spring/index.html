动态代理-cglib

cglib的实现原理：
在内存里构造一个目标对象的子类对象，
返回一个目标对象的子类对象的代理对象。

向项目中添加jar包:

  • asm-4.2.jar
  • cglib-3.1.jar

总结

在Spring的AOP编程中，

如果加入容器的目标对象实现了接口，使用JDK动态代理

如果加入容器的目标对象没有实现接口，使用cglib动态代理

内部类

$是内部类的标志

一个.java文件可以编译生成多个.class文件，数量取决于.java文件中定义的class类的数量

一个.java文件只能有一个public class

如果不是内部类，编译生成类名.class，如果是内部类，生成外部类$内部类.class
