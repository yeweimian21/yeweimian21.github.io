where

添加where子句。

如果where子句为空，则不添加where子句。

如果where标签后面直接连接and, or关键字，则and, or关键字会被去掉。

if

判断条件是否满足，满足条件则拼接sql，不满足条件则不拼接sql

choose/when/otherwise

相当于if/if else/else

set

用于update set子句，更新满足条件的字段

foreach

循环遍历数组或者集合。
